Ayuda SAC2


Introducción



El Sistema Adquirente Carnet SAC2, que en adelante y para fines prácticos será llamado CARNET SAC2, fue diseñado para el control operativo en Reportes Y Comparaciones de las transacciones adquirientes y emisoras de un Banco.



Por lo tanto esta provisto de un sistema alto de seguridad para la información de las transacciones generadas de acuerdo a los estándares de seguridad de Prosa.

Provee de una interfase Web al usuario, automatizando los procesos involucrados en la recuperación, interpretación y generación de reportes   de la  información de las transacciones generadas.



La aplicación Sistema Adquirente Carnet SAC2 fue desarrollada para poder ser accedida  desde cualquier terminal con el servicio de Internet y que cuente con un  explorador Web ya que esta publicada en Internet.



Una vez denotado lo anterior podemos decir que con el propósito de contar con las ventajas que ofrece el Internet, la aplicación CARNET SAC2  establece un funcionamiento estándar para todas las terminales que cuenten con este servicio y tengan un  explorador; definiendo  procesos genéricos representados a través de objetos Web.
